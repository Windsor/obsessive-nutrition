   - Click "Generate Video"
   - Wait 10-30 seconds
   - Go to "Video Library" tab
   - Click "Download"

🎓 TIPS FOR BEST RESULTS

QUIZ CREATION:

• Use clear, concise answers

• With TTS: 30-60 seconds (depends on API speed)
• Generate during off-peak hours for faster TTS

================================================================================
🆘 TROUBLESHOOTING
================================================================================

"PORT 5000 IN USE":
→ Edit app.py, change port to 5001
→ Or disable AirPlay Receiver (Mac)

"FFMPEG NOT FOUND":
→ Mac: brew install ffmpeg
→ Ubuntu: sudo apt install ffmpeg

"VIDEO GENERATION FAILS":
→ Check videos/ and frames/ directories exist
→ Check write permissions
→ View terminal logs for errors

"DATABASE ERROR":
→ Delete quiz_generator.db
→ Run: python3 -c "from app.database import init_db; init_db()"

"NO AUDIO IN VIDEOS":
→ Normal if no API key configured
→ Add ElevenLabs API key in Settings tab
