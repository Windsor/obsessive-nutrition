### New Video Flow (per question)
1. Show question (+ choices for MC) + TTS reads it → duration matches TTS length (min 3s)
2. Same frame + countdown overlay in corner → silent → duration = countdown_seconds
3. Answer reveal (correct choice highlighted green, others dimmed) + TTS says answer → duration matches TTS length
