# Content Templates — The Portugal Brief

Pre-built article structures for common story types. Use these in cron job prompts or manual publishing.

---

## 1. Breaking News

**Tags:** news, breaking
**Tone:** Urgent, factual, concise
**Structure:**
```
HEADLINE: [What happened] — clear, specific, no clickbait
LEAD (1 paragraph): Who, what, when, where. Most important fact first.
CONTEXT (1-2 paragraphs): Why this matters. Background for expats.
DETAILS (2-3 paragraphs): Supporting info, quotes, numbers.
WHAT'S NEXT (1 paragraph): Expected developments, timeline.
```

---

## 2. Daily Briefing

**Tags:** briefing
**Tone:** Conversational, scannable, informative
**Structure:**
```
HEADLINE: "[Morning/Afternoon] Update — [Day], [Date]"
INTRO: One-line summary of the day's tone.
STORIES (5-8 items):
  Each: Bold headline + 2-3 sentence summary with key facts.
  Order: Most important → least important.
SIGN-OFF: Brief, friendly closing.
```

---

## 3. Explainer / Guide

**Tags:** guides, [topic]
**Tone:** Helpful, clear, practical
**Structure:**
```
HEADLINE: "[Topic] in Portugal: What Expats Need to Know"
INTRO: Why this matters to English-speaking residents/visitors.
SECTION 1-4: Subheaded sections covering different aspects.
  Each: Clear explanation + practical advice + key numbers.
KEY TAKEAWAYS: Bullet list of 3-5 main points.
RESOURCES: Links to official sites (when available).
```

---

## 4. Economy / Markets

**Tags:** economy, [sub-topic]
**Tone:** Analytical, data-driven, accessible
**Structure:**
```
HEADLINE: [Key metric/change] — implications for Portugal
LEAD: The headline number and what it means.
DATA (1-2 paragraphs): Key figures, comparisons (EU average, previous period).
IMPACT (1-2 paragraphs): What this means for residents, businesses, property.
OUTLOOK (1 paragraph): Expert views, government response, expectations.
```

---

## 5. Politics

**Tags:** politics, [sub-topic]
**Tone:** Balanced, explanatory, no editorializing
**Structure:**
```
HEADLINE: [What happened or is proposed]
LEAD: Key political development.
CONTEXT: Political landscape, party positions.
DETAILS: Quotes, voting results, proposed legislation.
IMPACT: How this affects daily life, business, or expats.
REACTIONS: Different perspectives.
```

---

## 6. Lifestyle / Culture

**Tags:** lifestyle OR culture, [sub-topic]
**Tone:** Warm, engaging, personal
**Structure:**
```
HEADLINE: Engaging, specific, human-interest angle
HOOK: Interesting opening that draws in.
STORY (3-4 paragraphs): The main content — events, trends, traditions.
EXPAT ANGLE: How this relates to the English-speaking community.
PRACTICAL: If applicable — dates, locations, costs, tips.
```

## Image Selection Rules

1. **ALWAYS** use images from `image-library.md` or verified Portugal-specific Unsplash URLs
2. **NEVER** use generic stock photos of other countries
3. Match image to story topic (politics→parliament, economy→commerce, lifestyle→street scene)
4. Briefings use Lisbon/Porto cityscapes
5. If no suitable image, use a generic Portugal landscape

## Style Guide

- **Voice:** Informative, slightly conversational, never sensationalist
- **Audience:** English-speaking expats, digital nomads, investors in Portugal
- **Names:** Full name on first mention, surname after
- **Numbers:** Spell out under 10, use figures for 10+; always include € symbol
- **Dates:** Day Month format (19 February), not US format
- **Portuguese terms:** Italicize on first use, provide translation
